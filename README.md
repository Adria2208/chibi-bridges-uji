# About
Videogame demo made in Unity 6000.3.0f1 as an assignment for my Project I class at Jaume I University.

## List of features

Sophisticated 2D movement and jump mechanics with parametrized variables such as:
- Horizontal movement 
    - Max speed
    - Acceleration 
    - Deceleration
- Jump
    - Force 
    - Jump Hang
    - Jump buffer
    - Variable jump time
- Fall
    - Fast fall multiplier
    - Max fall speed
    - Coyote time
